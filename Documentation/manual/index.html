The sQuezzle Editor

sQuezzle started as a university project. My intention was to implement some basic feachers
of Gnu Emacs using only C++ and Qt. For this reason almost everything looks and feels like the Emacs editor.

sQuezzle uses a modified version of the GNU Emacs' documentation.

Emacs is the extensible, customizable, self-documenting real-time display editor.
The homepage for GNU Emacs is at http://www.gnu.org/software/emacs/.
Join the society for free software advocates on Free Software Foundation.
